# Video Carousel

Add a Video Carousel section to your Shopify store to display multiple videos in a clean, navigable carousel format directly on your storefront.

## Overview

The Video Carousel section lets you embed and showcase multiple videos in a horizontal carousel on your Shopify pages. Whether you need to display product demos, brand stories, or how-to tutorials, this section makes it easy to present video content in a polished, organized layout that works on any Shopify theme.

## Features

- Multi-video horizontal carousel layout
- Supports YouTube and Vimeo video embeds
- Compatible with any Shopify theme (Free or Paid)
- No paid app required
- Navigation controls for easy browsing
- Optional video title and description per slide
- Fully responsive for mobile and desktop
- Easy to set up through the Shopify theme customizer

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **Video Carousel** section to your desired page template
4. Add video blocks, each with a YouTube or Vimeo URL
5. Save and publish your changes

## Customization

- **Videos:** Add, remove, or reorder video blocks via the theme customizer
- **Video URL:** Enter a YouTube or Vimeo link for each video block
- **Captions:** Add an optional title or description per video
- **Slides per view:** Set how many videos are visible at a time
- **Navigation:** Enable or disable carousel arrows and indicators
- **Section heading:** Add an optional title above the carousel
