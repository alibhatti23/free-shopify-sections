# FAQ Section

Add a professional FAQ (Frequently Asked Questions) section to your Shopify store, with guides for both a dedicated FAQ page and an embeddable FAQ section that can be added anywhere on your site.

## Overview

The FAQ section provides an accordion-style question and answer layout for your Shopify storefront. Use it on a standalone FAQ page or embed it within product, collection, or landing pages to address common customer questions and reduce support inquiries.

## Features

- Accordion-style expand/collapse FAQ items
- Works as a full FAQ page or an embeddable section
- Compatible with any Shopify theme (Free or Paid)
- Add unlimited question and answer blocks
- Smooth open/close animation
- Fully responsive for all screen sizes
- No paid app required
- SEO-friendly structured content

## Getting Started

1. Copy the section Liquid file into your theme's `sections/` directory
2. Navigate to **Online Store → Themes → Customize** in your Shopify admin
3. Add the **FAQ** section to your desired page template (FAQ page or any other)
4. Add question and answer blocks via the block settings
5. Save and publish your changes

## Customization

- **Questions & answers:** Add, remove, or reorder FAQ items via the theme customizer
- **Section heading:** Set a title displayed above the FAQ list
- **Expand behavior:** Configure whether one or multiple items can be open at once
- **Styling:** Adjust colors, fonts, and borders to match your store brand
- **Icons:** Customize the expand/collapse indicator icon
