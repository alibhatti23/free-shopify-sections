# Add Custom Font in Shopify

Learn how to add custom fonts to any Shopify theme — including Dawn, Horizon, and more — without touching a single line of CSS, using a free section with no paid apps required.

## Overview

This free Custom Font section lets you upload and apply any font to your Shopify store through a simple toggle-based interface. Control which elements use your custom font (headings, menus, links, body text) directly from the theme customizer — no coding knowledge needed.

## Features

- Apply custom fonts without editing any CSS
- Works on ANY Shopify theme (Dawn, Horizon, Atelier, Ritual, and more)
- Upload custom font files directly to Shopify
- Apply fonts to specific elements: H1, H2, menus, links, and more
- Enable or disable fonts instantly with a simple toggle
- No paid apps, no newsletter sign-ups, no coding required
- Supports WOFF, WOFF2, TTF, and OTF font formats

## Getting Started

1. Upload your font files to **Online Store → Themes → Assets**
2. Copy the section Liquid file into your theme's `sections/` directory
3. Navigate to **Online Store → Themes → Customize** in your Shopify admin
4. Add the **Custom Font** section to your theme layout
5. Select which elements to apply the font to and toggle it on
6. Save and publish your changes

## Customization

- **Font file:** Upload any WOFF, WOFF2, TTF, or OTF font file
- **Target elements:** Choose H1, H2, body, menus, links, buttons, and more
- **Font weight & style:** Configure weight and italic variants
- **Toggle:** Instantly enable or disable the custom font without removing the section
